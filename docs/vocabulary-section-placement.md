## Metadata Moves

The following sets were imported as high-frequency language but now have more
useful browsing metadata:

| Vocabulary set                    | Previous section        | Current section                    | Reason                                                    |
| --------------------------------- | ----------------------- | ---------------------------------- | --------------------------------------------------------- |
| Countries                         | High-frequency language | Local area, holiday and travel     | Countries are most useful beside travel/place vocabulary. |
| Areas, mountains, seas and places | High-frequency language | Local area, holiday and travel     | These are place/geography words.                          |
| Nationalities                     | High-frequency language | Identity and culture               | Nationality is identity/person-description vocabulary.    |
| Continents                        | High-frequency language | International and global dimension | Continents fit global/world-topic browsing.               |

Everything else currently in Section 1 is either structural language, numbers,
time language, question words, connective language, conventions, or dialogue
language, so it should stay in high-frequency language.
